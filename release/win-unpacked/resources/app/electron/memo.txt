npm install find-process tree-kill
npm install --save-dev electron-builder
npm install pdf-lib

2 各種JS,html.cssを作成

3 package.json
"main": "electron/main.js"
"scripts": {
  "start": "node server03.js",
  "electron": "electron ."
}

4 起動方法
npm run electron
npm run dist 
→dist/win-unpacked/BoatRaceManager.exe　→exe化したもの
